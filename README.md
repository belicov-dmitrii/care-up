# Intelligent Medication Assistant

A web application for organizing medication intake, tracking prescriptions, storing a personal medicine cabinet, and analyzing medical data.

## Overview

This project is built with Next.js, React, TypeScript, and MUI.  
It is designed as a modern medical assistant interface focused on:

- medication schedule management
- prescription and analysis storage
- intake reminders and notifications
- medicine cabinet tracking
- basic medical data visualization

## Main Features

- User authentication
- Medication and prescription management
- Personal medicine cabinet
- Schedule creation for medication intake
- Analysis upload and processing
- Notifications and reminders
- Responsive UI for desktop and mobile

## Tech Stack

- Next.js
- React
- TypeScript
- MUI
- SCSS / CSS Modules
- OneSignal for push notifications
- Google Maps / location-based features for nearby pharmacies
- Local JSON / API-based data handling depending on module
